# Student-Dropout-Prediction
This project aims to analyze student data and build machine learning models to predict whether a student is likely to Dropout, remain Enrolled, or Graduate. The goal is to support educational institutions in identifying at-risk students early, enabling timely interventions to improve academic success and retention rates.

Educational dropout is a critical challenge faced by universities worldwide. By leveraging student demographic, academic, and socio-economic data, this project builds predictive models to classify student outcomes. The study includes data exploration, feature analysis, model building, and model evaluation to identify key factors contributing to student dropout.
